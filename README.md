- Interactive Streamlit application
- Dynamic question flow based on change type
- Supports:
  - Script Execution
  - Stored Procedure Execution
  - SSIS Package Execution
  - Variable ID Execution
- Displays all collected inputs at the end
- Easy to extend with AI or automation

---
###  Install dependencies
pip install -r requirements.txt

###  Run the app
python -m streamlit run app.py
